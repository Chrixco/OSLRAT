Getting Started

OSLRAT is designed to be accessible to users with varying levels of technical expertise. Whether you're using the web interface or the Python library, this guide will help you get up and running quickly.

Prerequisites

  • Python 3.8 or higher (for Python API)
  • Modern web browser (for web interface)
  • Basic understanding of sea-level rise concepts

Installation

Using pip (Recommended)

Install OSLRAT using pip:

bash 
pip install oslrat

From Source

For the latest development version:

bash 
git clone https://github.com/yourusername/OSLRAT.git
cd OSLRAT
pip install -e .

Verify Installation

python 
import oslrat
print(oslrat.__version__)

Quick Start

Here's a simple example to get you started with OSLRAT:

python 
import oslrat

# Initialize the assessment tool
slr = oslrat.SeaLevelRise()

# Load data for a specific location
location = slr.load_location(lat=37.7749, lon=-122.4194)  # San Francisco

# Get projections for different scenarios
projections = location.get_projections(
    scenarios=['SSP1-2.6', 'SSP2-4.5', 'SSP5-8.5'],
    years=[2030, 2050, 2100]
)

# Visualize the results
location.plot_projections(projections)

# Export data
projections.to_csv('san_francisco_slr.csv')

Data Visualization

OSLRAT provides multiple visualization options for different use cases.

Interactive Maps

python 
# Create an interactive map
map_view = oslrat.InteractiveMap(
    center=(37.7749, -122.4194),
    zoom=11
)

# Add sea-level rise layer
map_view.add_slr_layer(
    year=2100,
    scenario='SSP5-8.5',
    opacity=0.7
)

# Add vulnerable infrastructure
map_view.add_infrastructure_layer()

# Display the map
map_view.show()

Time Series Charts

python 
# Plot time series for multiple scenarios
chart = oslrat.TimeSeriesChart()
chart.plot(
    data=projections,
    scenarios=['SSP1-2.6', 'SSP2-4.5', 'SSP5-8.5'],
    confidence_intervals=True,
    style='publication'
)
chart.save('timeseries.png', dpi=300)

Risk Assessment

Conduct comprehensive risk assessments for coastal areas.

python 
# Define study area
study_area = oslrat.StudyArea.from_bounds(
    north=37.8324,
    south=37.7074,
    east=-122.3549,
    west=-122.5194
)

# Run risk assessment
assessment = oslrat.RiskAssessment(study_area)
results = assessment.analyze(
    scenarios=['SSP2-4.5', 'SSP5-8.5'],
    years=[2050, 2100],
    include_population=True,
    include_infrastructure=True,
    include_economic_impact=True
)

# Generate report
report = results.generate_report(
    format='pdf',
    include_maps=True,
    include_recommendations=True
)
report.save('risk_assessment_report.pdf')

Note on Uncertainty

All projections include uncertainty ranges. Always consider the full range of possible outcomes in your planning.

Scenario Planning

Compare different emission scenarios and adaptation strategies.

python 
# Create scenario comparison
scenarios = oslrat.ScenarioPlanner()

# Add baseline (no adaptation)
scenarios.add_scenario(
    name='No Adaptation',
    slr_scenario='SSP5-8.5',
    adaptation_measures=[]
)

# Add adaptation scenarios
scenarios.add_scenario(
    name='Seawall Protection',
    slr_scenario='SSP5-8.5',
    adaptation_measures=[
        oslrat.Seawall(height=3.0),
        oslrat.BeachNourishment(interval=5)
    ]
)

scenarios.add_scenario(
    name='Managed Retreat',
    slr_scenario='SSP5-8.5',
    adaptation_measures=[
        oslrat.ManagedRetreat(trigger_level=1.5)
    ]
)

# Compare scenarios
comparison = scenarios.compare(
    metrics=['cost', 'protection_level', 'co2_reduction'],
    time_horizon=2100
)

# Visualize comparison
comparison.plot_comparison()

Export & Reports

Export data and generate reports in various formats.

Data Export

python 
# Export to various formats
results.to_csv('data.csv')
results.to_excel('data.xlsx')
results.to_geojson('data.geojson')
results.to_netcdf('data.nc')

Report Generation

python 
# Generate comprehensive report
report = oslrat.Report()
report.add_section('Executive Summary', auto_generate=True)
report.add_section('Methodology', include_citations=True)
report.add_section('Results', include_visualizations=True)
report.add_section('Recommendations', risk_based=True)

# Export report
report.export('assessment_report.pdf', style='scientific')
report.export('assessment_report.html', interactive=True)

API Reference

Core Classes

oslrat.SeaLevelRise

Main class for sea-level rise analysis and projections. 

SeaLevelRise(
    data_source='auto',
    cache_dir=None,
    verbose=True
)
Parameters:
  • data_source (str): Data source to use. Options: 'auto', 'ipcc', 'noaa'
  • cache_dir (str): Directory for caching downloaded data
  • verbose (bool): Enable verbose output

oslrat.Location

Represents a specific geographic location for analysis. 

Location(
    latitude,
    longitude,
    name=None,
    elevation=None
)

oslrat.RiskAssessment

Conducts comprehensive coastal risk assessments. 

RiskAssessment(
    study_area,
    dem_resolution='30m',
    include_subsidence=True
)

Troubleshooting

Common Issues

Installation fails with dependency errors

Solution: Try creating a fresh virtual environment and installing with: 

python -m venv oslrat-env
source oslrat-env/bin/activate  # On Windows: oslrat-env\Scripts\activate
pip install --upgrade pip
pip install oslrat

Data download is very slow

Solution: Consider using a different data source or mirror: 

slr = oslrat.SeaLevelRise(
    data_source='noaa',  # Try alternative source
    cache_dir='/path/to/large/disk'  # Use disk with more space
)

Memory errors with large study areas

Solution: Process data in smaller chunks or reduce resolution: 

assessment = oslrat.RiskAssessment(
    study_area,
    dem_resolution='90m',  # Lower resolution
    chunk_size=1000  # Process in chunks
)
